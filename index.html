Teabuzz Data Deletion Request

Teabuzz allows users to request deletion of specific data without deleting their account.

How to Request Data Deletion

  1. Email blackishtechnologies@gmail.com
  2. Clearly describe the data you want deleted
  3. Requests are processed within 7 days

Types of Data You Can Request to Delete

  • User posts and uploaded content
  • Profile details
  • Activity and usage data

Data Retention

Some limited data (such as security logs) may be retained for up to 30 days for fraud prevention and legal compliance.
